# 🏦 Bank Management System

A console-based Bank Management System developed in Python using Object-Oriented Programming (OOP) principles and file handling. This project simulates basic banking operations and demonstrates how account data can be managed using text files.

---

## 🚀 Features

* 👤 Create New Account
* 💰 Deposit Money
* 💸 Withdraw Money
* 📄 View Account Details
* 🗑️ Delete Account
* ✅ Duplicate Account Validation
* ✅ Account Existence Validation
* 💾 Persistent Data Storage Using File Handling

---

## 🛠️ Technologies Used

* Python
* Object-Oriented Programming (OOP)
* File Handling
* Static Methods
* Menu-Driven Application Design
